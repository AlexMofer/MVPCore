MVP Core
========

介绍
---

MVP设计模式基础核心，与平台无关，便于进一步实现Android或者鸿蒙的MVP设计模式。

使用
---

**引用:**
```
dependencies {
    ...
    implementation 'io.github.alexmofer.mvp:mvp-core:1.0.1'
    ...
}
```
